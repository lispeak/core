# Главные формы в языках

Пол: Мужской/женский (он/она)
Число: Единственное/множественное (я/мы)
Время: Прошлое/Настоящее/Будущее (делал/делает/будет дел)
Статус: Процесс/Завершенность (сделал/делал)
Определённость: Определённый / неопределённый (the/a)
Лицо: Первое лицо / Второе лицо / Третье лицо (я/ты/он/она/оно)
Залог: Активный залог / Пассивный залог (делать/делаться)
Степень сравнения: Положительная / Отрицательная / Нейтральная (быстрый/быстрее/самый быстрый)
